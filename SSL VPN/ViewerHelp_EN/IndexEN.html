1 Overview
Neusoft NetEye SSL VPN Client (hereinafter referred to as SSL VPN Client) is a VPN software based on the SSL protocol. It can be installed on iOS 10.X and later versions. SSL VPN is a simple and secure technology which can be used by remote users to access sensitive data of their companies.
Neusoft NetEye series security devices include Neusoft NetEye Firewall (FW), Neusoft NetEye Integrated Security Gateway (NISG), and Neusoft NetEye Integrated Security Gateway Intrusion Prevention System (NISG-IPS). With SSL VPN Client, remote users can establish an SSL VPN tunnel with a Neusoft NetEye series security device (hereinafter referred to as “the SSL VPN server” or “the server”) to secure data transmission. After successfully establishing a VPN connection with the SSL VPN server, remote users can access the protected resources of their companies, such as Web servers, FTP servers, and Mail servers.
